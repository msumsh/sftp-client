### Подключение клиента к серверу
#### Позитивные сценарии
- Подключение успешно;
- Файл успешно загружен с сервера
- Изменённый файл успешно загружен обратно на сервер;
- Соединение корректно закрывается после завершения работы;
#### Негативные сценариии
- Сервер недоступен;
- Указан неверный порт;
- Указан неверный хост;
- Указан неверный логин;
- Указан неверный пароль;
- На сервере отсутствует указанный файл;
